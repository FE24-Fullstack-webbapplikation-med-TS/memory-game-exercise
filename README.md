# Statiska typer

## Övningar

**Memory game**
Du ska skapa spelet memory och detta ska göras med hjälp av Typescript. Du hittar i här en dist-mapp med HTML och CSS samt en tom JS-fil och sen i din src-mapp en tom `index.ts` där själva logiken för spelet ska ligga. I din HTML-fil finns all HTML som du behöver och detsamma för CSS, en CSS-klass om heter `.flip` kan användas för att flippa kortet och visa baksidan.

Du behöver även konfigurera `tsconfig.json`.
